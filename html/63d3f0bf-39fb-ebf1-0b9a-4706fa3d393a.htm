Shader Class

Encapsulates a shader program and attached shaders. Errors are stored to an internal log, which can be exported with GetErrorLog.

Ensure that ProgramCreatedSuccessfully returns true before rendering to avoid crashes.

ProgramCreatedSuccessfully
true when the link status is ok and all attached shaders compiled. If false, rendering with this shader will most likely cause an AccessViolationException.

The status is updated with each call to LoadShader(String, ShaderType, String) or AttachShader(Int32, ShaderType).
