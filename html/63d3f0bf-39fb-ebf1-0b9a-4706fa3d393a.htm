Shader Class

Encapsulates a shader program and attached shaders. Errors are stored to an internal log, which can be exported with GetErrorLog.

Ensure that ProgramCreatedSuccessfully returns true before rendering to avoid crashes.

ProgramCreatedSuccessfully
Returns true when the link status is ok and all attached shaders compiled. If false, rendering with this shader will most likely cause a crash.

The status is only updated the first time ProgramCreatedSuccessfully is called, so there is little cost in checking this method frequently.
