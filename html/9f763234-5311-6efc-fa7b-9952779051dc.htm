Shader Methods

The Shader type exposes the following members.

Methods
  NameDescription
AttachShader
Attaches shaderId and links the program. The value for LinkStatusIsOk is updated.
CreateGlShader
Creates and compiles a new shader from shaderSource. Returns the ID created by GL.CreateShader(). Shaders can be attached with AttachShader(Int32, ShaderType, String)
DisableVertexAttributes
Disables the vertex attribute arrays for all active attributes for the currently bound vertex array object. Ensures that extra attributes aren't enabled, which causes crashes on Geforce drivers.
EnableVertexAttributes
Enables the vertex attribute arrays for all active attributes for the currently bound vertex array object. Ensures that extra attributes aren't enabled, which causes crashes on Geforce drivers.
Equals
Determines whether the specified object is equal to the current object.
(Inherited from Object.)
GetAttribLocation
Gets the attribute's location. Potentially faster than GL.GetAttribLocation due to prestored locations.
GetErrorLog
Gets the error log containing hardware info, version number, compilation/linker errors, and attempts to initialize invalid uniform or vertex attribute names.
GetHashCode
Serves as the default hash function.
(Inherited from Object.)
GetProgramBinary
Gets the compiled program binary for the program Id. This method should be called after the shaders are loaded and the program is linked. Hardware or software changes may cause compatibility issues with the program binary.
GetType
Gets the Type of the current instance.
(Inherited from Object.)
GetUniformLocation
Gets the uniform's location. Potentially faster than GL.GetUniform due to prestored locations.
LoadProgramBinary
Loads the entire program from the compiled binary and format generated by GetProgramBinary(BinaryFormat). The value returned by LinkStatusIsOk is updated.

Hardware or software changes may cause compatibility issues with the program binary. If program creation fails with precompiled binaries, resort to compiling the shaders from source.

LoadShader
Compiles and attaches a shader from shaderSource. linkStatusIsOk is updated.
SetBoolToInt
Converts value to an int and sets an int uniform. true = 1. false = 0. uniform variable. Logs invalid names.
SetFloat(String, Single)
Sets a float uniform variable. Logs invalid names.
SetFloat(String, Single)
Sets all values for a float[] uniform variable. Logs invalid names.
SetInt(String, Int32)
Sets an int uniform variable. Logs invalid names.
SetInt(String, Int32)
Sets all values for an int[] uniform variable. Logs invalid names.
SetMatrix4x4
Sets a mat4 uniform variable. Logs invalid names.
SetTexture
texture is bound to textureUnit before setting the uniform. Logs invalid names.
SetUint(String, UInt32)
Sets a uint uniform variable. Logs invalid names.
SetUint(String, UInt32)
Sets all values for a uint[] uniform variable. Logs invalid names.
SetVector2(String, Vector2)
Sets vec2 uniform variable. Logs invalid names.
SetVector2(String, Single, Single)
Sets vec2 uniform variable. Logs invalid names.
SetVector3(String, Vector3)
Sets vec3 uniform variable. Logs invalid names.
SetVector3(String, Single, Single, Single)
Sets vec3 uniform variable. Logs invalid names.
SetVector4(String, Vector4)
Sets vec4 uniform variable. Logs invalid names.
SetVector4(String, Single, Single, Single, Single)
Sets vec4 uniform variable. Logs invalid names.
ToString
Returns the type of OpenGL object and the integer ID generated by OpenGL. Example: "ShaderProgram ID: 1"
(Inherited from GLObject.)
