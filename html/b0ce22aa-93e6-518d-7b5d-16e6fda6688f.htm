Properties

ColorAttachments
All color attachment textures. This list may be empty if 0 is specified for the color attachment count. Attempting to modify the textures in this list may result in unexpected behavior.
FramebufferTarget
The target which Id is bound when calling Bind.
Height
All attached textures, renderbuffers, etc are resized when set. The framebuffer's contents will not be preserved when resizing.
Id
The value generated by GL.GenFramebuffer(). Do not attempt to bind Id when the object has become unreachable.
