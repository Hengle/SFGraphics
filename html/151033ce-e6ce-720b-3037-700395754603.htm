VectorUtilsGenerateTangentBitangent Method

Generates a tangent vector tangent and a bitangent vector bitangent for a triangle face. If the three vertices have the same UVs or position, tangent is set to (1, 0, 0) and bitangent is set to (0, 1, 0). This prevents black shading artifacts.

tangent and bitangent should be added to the existing tangent and bitangent value for each vertex in the triangle. Normalizing the final sum averages the tangents and bitangents for smoother results.
