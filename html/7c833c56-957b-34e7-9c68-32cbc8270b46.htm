The VectorTools type exposes the following members.

Methods
  NameDescription
CalculateNormal
Calculates the face normal of a triangle. The result is not normalized. A triangle facing the camera will have a positive normal when the verts are ordered counter-clockwise.
GenerateTangentBitangent
Generates a tangent vector tangent and a bitangent vector bitangent for a triangle face. If the three vertices have the same UVs or position, tangent is set to (1, 0, 0) and bitangent is set to (0, 1, 0). This prevents black shading artifacts.

tangent and bitangent should be added to the existing tangent and bitangent value for each vertex in the triangle. Normalizing the final sum averages the tangents and bitangents for smoother results.

GetDegrees
Converts radians to degrees.
GetRadians
Converts degrees to radians.
Orthogonalize
Uses the Gran-Schmidt method for orthogonalizing a vector to another vector. The resulting vector is normalized.

Ex: Vector3 tanOrthoToNrm = Orthogonalize(tan, nrm);
