Classes
  ClassDescription
Texture
Encapsulates the state for an OpenGL texture object. To support texture types other than Texture2D and TextureCubeMap, inherit from this class and add the necessary additional methods.

Avoid creating textures manually to prevent issues with textures being deleted by DeleteUnusedGLObjects.

Texture2D
A Texture class for TextureTarget.Texture2D textures. Provides constructors to initialize the image data from bitmaps or byte arrays in the specified PixelInternalFormat.
TextureCubeMap
A Texture class for TextureTarget.TextureCubeMap textures. Provides a constructor for initializing the cube map faces from faces arranged vertically in a single bitmap.
