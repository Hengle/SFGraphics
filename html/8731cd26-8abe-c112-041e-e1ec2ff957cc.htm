Classes
  ClassDescription
DepthTexture
A simple texture for storing depth information. The texture can be attached to a Framebuffer object for shadow mapping and other effects.
MipmapLoading
Provides methods for loading mipmaps for OpenGL textures from arrays of image data. The arrays can be of any value type. As long as the correct image format information is used, OpenGL will still interpret the data correctly. Make sure to bind the texture before calling these methods.
Texture
Encapsulates the state for an OpenGL texture object. To support texture types other than Texture2D and TextureCubeMap, inherit from this class and add the necessary additional methods.
Texture2D
A TextureTarget.Texture2D texture that supports mipmaps.
TextureCubeMap
A TextureTarget.TextureCubeMap texture that supports mipmaps.
