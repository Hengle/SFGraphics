SFGraphics.GLObjects Namespace

 
Classes
  ClassDescription
BufferObject
Data can be read from and written to the buffer using any value type.

This class does not permanently store the data used for initialization.

Framebuffer
Encapsulates an OpenGL framebuffer, including any attached color or depth attachments.
GLObject
Provides a simpler and less error prone way to deal with the state and data associated with OpenGL objects.
Renderbuffer
Encapsulates a renderbuffer object, which are used exclusively as attachments for Framebuffer objects. Renderbuffers support multisampling.
VertexArrayObject
Encapsulates an OpenGL vertex array object. Vertex array objects cannot be shared between contexts.
