Calculates the image size parameter for GL.CompressedTexImage. The image size should be recalculated for each mip level.

IsCubeMapTarget

IsDepthFormat
Determines if format is a valid format for a DepthTexture.
IsGenericCompressedFormat
Determines if format is a generic compressed format, which are implementation dependent and do not work for loading texture data.
