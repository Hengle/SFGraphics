GLObjectManager Class

GLObject and all subclasses use reference counting to determine what OpenGL objects can be safely deleted.

Call DeleteUnusedGLObjects with a valid OpenGL context current to delete objects with no references.

The reference count may not be decremented until after the context is destroyed. Call WaitForPendingFinalizers prior to DeleteUnusedGLObjects if more immediate cleanup is desired.
